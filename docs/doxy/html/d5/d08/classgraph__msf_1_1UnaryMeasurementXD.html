Detailed Description

template<class MEASUREMENT_TYPE, int DIM>
class graph_msf::UnaryMeasurementXD< MEASUREMENT_TYPE, DIM >

Class to represent x-dimensional unary measurement in the graph.

Description: This class is used to represent x-dimensional unary measurement in the graph.

Template Parameters
MEASUREMENT_TYPEThe type of the unary measurement. E.g. Eigen::Vector3d for a 3D position measurement.
DIMThe dimension of the unary measurement. E.g. '3' for a 3D position measurement.
