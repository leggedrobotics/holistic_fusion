Detailed Description

template<class GTSAM_MEASUREMENT_TYPE, UnaryExpressionType TYPE, char CALIBRATION_CHAR>
class graph_msf::GmsfUnaryExpression< GTSAM_MEASUREMENT_TYPE, TYPE, CALIBRATION_CHAR >

base class for unary expressions. Unary expressions are used to represent unary factors in the factor graph. It optionally supports the HolisticGraph paradigm to align different measurements. It optionally supports extrinsic calibration of the sensor.

Template Parameters
GTSAM_MEASUREMENT_TYPEType of the measurement (e.g. gtsam::Pose3).
TYPEType of the unary expression (e.g. UnaryExpressionType::Absolute).
CALIBRATION_CHARCharacter representing the calibration type (e.g. 'c').
