Detailed Description

template<class MEASUREMENT_TYPE, int DIM>
class graph_msf::UnaryMeasurementXDAbsolute< MEASUREMENT_TYPE, DIM >

Class to represent x-dimensional absolute unary measurement in the graph.

Description: This class is used to represent x-dimensional absolute unary measurement in the graph.

Template Parameters
MEASUREMENT_TYPEThe type of the unary measurement. E.g. Eigen::Vector3d for a 3D position measurement.
DIMThe dimension of the unary measurement. E.g. '3' for a 3D position measurement.

Parameters
measurementNameName of the measurement.
measurementRateRate of the measurement. Mostly used for checking in the core whether all is working correctly.
sensorFrameNameName of the sensor frame.
sensorFrameCorrectedNameName of the corrected sensor frame.
robustNormRobust norm to be used for the measurement.
measurementTypeEnumType of the measurement.
timeStampTime stamp of the measurement. Argument of parent class.
covarianceViolationThresholdThreshold for the covariance violation. Argument of parent class.
fixedFrameNameName of the fixed frame. This is the frame w.r.t. which the ABSOLUTE measurement is defined.
worldFrameNameName of the world frame. This is needed for checking correctness of provided arguments.
initialSe3AlignmentNoiseInitial noise for the SE3 alignment. If not provided, fixed frame has to be the world frame (as no alignment can be performed).
se3AlignmentRandomWalkRandom walk for the SE3 alignment. If not provided or set to zero, no random walk is used but alignment is treated as global variable.
