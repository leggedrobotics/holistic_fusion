10enum class DynamicVariableTypeEnum { Global, RefFrame, Landmark };
11
12// Explanation:
13// 1) global variables are constant along the entire operation, hence they are not removed and will get the same gtsam key upon return
14// 2) reference frames can change, but are modelled according to random walk, hence they are only removed once a successor is initialized
15// 3) landmarks are initialized and removed as needed, hence they are removed once they are not needed anymore
16// Frames that are not needed but also not removed are set to be "inactive"
17
18// Robust Norm Container
